# Fluebilder

Bilder som vises i «Fishon»-fanen (topp 3 fluekort). Legg JPG-filer her med filnavn som
matcher slug-ene under. Mangler en fil, faller kortet pent tilbake til en nøytral plassholder.

Fluevalget er nå Lesja sone 7-spesifikt og rangeres dynamisk etter sesong + live-forhold
(se `LESJA_FLIES` i `scoring.js`). Bare topp 3 laster bilde, så bare de øverst rangerte
fluene trenger bildefil.
